# ❤️ Heart Disease AI Predictor

A premium, production-ready machine learning web app built using **Streamlit** to predict the risk of heart disease based on patient health parameters.

---

## 🚀 Features

* 🔮 Machine Learning-based prediction
* 🎨 Premium UI (Glassmorphism + Dark Theme)
* 📊 Risk Probability Score
* 🧠 Clean and user-friendly interface
* ✅ Input validation (no negative values allowed)
* ⚡ Fast and interactive performance

---

## 🛠️ Tech Stack

* Python 🐍
* Streamlit 🎈
* NumPy
* Pickle (Model Serialization)

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/heart-disease-predictor.git
cd heart-disease-predictor
```

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run the App

```
streamlit run app.py
```

---

## 🧠 Model Information

The model is trained using a heart disease dataset with features like:

* Age
* Sex
* Chest Pain Type
* Cholesterol
* Blood Pressure
* Heart Rate
* And more...

The trained model is saved as `model.pkl` and loaded into the Streamlit app for real-time predictions.

---

## 📊 How It Works

1. User enters patient details
2. Inputs are validated
3. Data is passed to the ML model
4. Prediction is generated
5. Risk score (%) is displayed

---

## 🔐 Input Validation

* Prevents negative values
* Ensures realistic medical inputs
* Improves prediction accuracy
